About Hujanio

Hujan.io is an all-in-one designed deployment tool that can be used to build a Hyper Converged Infrastructure (HCI) in an easy and user-friendly way. Created to fulfill GUI / UI needs that are not yet available in other deployment tools. Using MAAS (Metal as a Service) as baremetal management and kolla-ansible as the basis of deployment tools.
